# Harry Potter — Women & Magic (Quick Analysis)

This repository contains a short Jupyter notebook (Untitled4.ipynb) that performs a quick analysis of female characters and their magical dialogue in Harry Potter.

What you'll find
- `Untitled4.ipynb` — notebook with sample dataset, analysis functions, visualizations (wordcloud, network, bar charts, sentiment)
- `requirements.txt` — list of Python packages used

Run locally (recommended)
1. Create and activate a virtual environment (macOS / zsh):

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start Jupyter Lab or Notebook and open `Untitled4.ipynb`:

```bash
jupyter lab
# or
jupyter notebook
```

Notes
- The notebook includes a small sample dataset; replace it with your own CSV if needed.
- If you see errors about missing NLTK data, run the NLTK downloader inside the notebook: `nltk.download('vader_lexicon')`.
